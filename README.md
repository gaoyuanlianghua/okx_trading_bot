# OKX 交易机器人

## 项目简介

OKX 交易机器人是一个基于 PyQt5 和 WebSocket 的实时交易界面，提供了丰富的功能，包括市场数据监控、策略管理、交易执行等。该机器人支持实时数据更新、响应式布局、多种技术指标分析、策略性能分析等功能，为用户提供了一个直观、高效的交易工具。

## 系统架构

### 核心组件

- **API 客户端**：包括 OKX REST API 客户端和 WebSocket 客户端，负责与 OKX 交易所 API 进行通信
- **智能体系统**：包括市场数据智能体、订单智能体、风险智能体、策略智能体和协调智能体，负责系统的核心业务逻辑
- **事件总线**：实现组件间的消息传递和事件通知
- **GUI 界面**：基于 PyQt5 的用户界面，提供直观的操作体验
- **策略系统**：支持多种交易策略的创建、管理和执行

## 功能特点

### 核心功能

- **实时市场数据**：通过 WebSocket 连接获取实时市场数据，包括价格、涨跌幅、成交量等
- **仪表盘概览**：提供市场概览、账户概览、策略概览、交易概览、风险指标、持仓概览等卡片
- **响应式布局**：适配不同屏幕尺寸，自动调整布局方式
- **深色/浅色主题**：支持主题切换，提供舒适的视觉体验
- **技术指标分析**：支持 MA、RSI、MACD、BOLL、KDJ、CCI、ATR 等多种技术指标
- **策略管理**：支持创建、启动、停止、编辑、删除策略
- **策略性能分析**：生成详细的策略性能分析报告
- **数据导出**：支持导出市场数据为 CSV 和 Excel 格式
- **API 密钥加密存储**：使用 Fernet 加密算法安全存储 API 密钥
- **自动重连机制**：WebSocket 连接断开后自动尝试重连
- **系统健康检查**：定期检查系统状态，确保正常运行

### 界面特性

- **标签页动画**：标签页切换时的淡入淡出动画效果
- **拖拽功能**：支持拖拽调整仪表盘卡片位置
- **实时更新**：仪表盘数据每 5 秒自动更新
- **图表交互**：支持时间范围选择和技术指标切换

## 安装和配置

### 系统要求

- Python 3.7+

### 安装依赖

```bash
pip install -r requirements.txt
```

### 依赖项

- **GUI 界面**：PyQt5>=5.15.0
- **加密功能**：cryptography>=3.4.0
- **图表绘制**：matplotlib>=3.5.0
- **数据处理**：pandas>=1.3.0
- **系统资源监控**：psutil>=5.8.0
- **Excel 导出**：openpyxl>=3.0.0
- **HTTP请求**：requests>=2.28.0
- **异步HTTP请求**：aiohttp>=3.8.0
- **WebSocket通信**：websockets>=10.0.0
- **测试依赖**：pytest>=8.0.0, pytest-asyncio>=0.24.0, pytest-qt>=4.0.0
- **金融图表**：mplfinance>=0.12.9b0

### API 配置

1. 在 OKX 官网申请 API 密钥
2. 启动应用程序后，在连接管理部分输入 API Key、Secret 和 Passphrase
3. 选择环境（模拟盘或实盘）
4. 点击"连接"按钮连接 WebSocket

### 加密密钥

首次启动时，应用程序会自动生成加密密钥并保存在 `encryption_key.key` 文件中。请妥善保管此文件，不要分享给他人。

## 使用指南

### 连接管理

1. 在顶部的连接管理区域输入 API 密钥信息
2. 选择交易环境（模拟盘或实盘）
3. 点击"连接"按钮建立 WebSocket 连接
4. 连接成功后，连接状态会显示为"已连接"

### 市场数据

1. 在"市场数据"标签页查看实时市场数据
2. 使用搜索框搜索特定交易对
3. 选择时间周期和技术指标
4. 调整时间范围查看不同时间段的价格走势
5. 点击"导出CSV"或"导出Excel"按钮导出数据

### 交易与订单

1. 在"交易与订单"标签页进行交易操作
2. 选择交易对、方向、类型、价格和数量
3. 点击"执行交易"按钮执行交易
4. 在"批量交易"标签页输入多个交易指令
5. 在"条件单"标签页设置止盈止损条件
6. 在下方查看订单列表和账户信息

### 策略管理

1. 在"策略管理"标签页管理策略
2. 输入策略名称和类型，点击"创建策略"按钮
3. 在弹出的对话框中输入或粘贴策略代码
4. 选择策略，点击"启动策略"按钮启动策略
5. 点击"停止策略"按钮停止运行中的策略
6. 点击"编辑策略"按钮修改策略代码
7. 点击"删除策略"按钮删除策略
8. 选择策略，点击"生成报告"按钮生成策略性能分析报告

### 监控与分析

1. 在"监控与分析"标签页查看系统健康状态
2. 查看网络状态、CPU 和内存使用情况
3. 监控策略运行状态

## API 客户端

### REST API 客户端

- 支持所有 OKX REST API 端点
- 实现了请求签名和认证
- 支持模拟盘和实盘环境
- 包含请求限流和重试机制
- 支持响应数据缓存

### WebSocket 客户端

- 支持公共频道和私有频道
- 实现了自动重连机制
- 支持心跳检测
- 支持多种消息类型的处理
- 提供了简洁的订阅接口

## 智能体系统

### 市场数据智能体

- 负责获取和处理市场数据
- 管理数据缓存
- 提供数据查询接口

### 订单智能体

- 负责订单的创建、修改和取消
- 管理订单状态
- 提供订单查询接口

### 风险智能体

- 负责风险评估和管理
- 计算风险指标
- 提供风险控制建议

### 策略智能体

- 负责策略的加载和执行
- 管理策略状态
- 提供策略性能分析

### 协调智能体

- 协调各个智能体之间的通信
- 管理系统状态
- 处理系统级事件

## 策略系统

### 内置策略

- **移动平均线RSI策略**：结合移动平均线和RSI指标进行交易
- **MACD布林带策略**：结合MACD和布林带指标进行交易

### 策略开发

1. 继承 `BaseStrategy` 类
2. 实现 `execute` 方法
3. 在策略管理界面创建和部署策略

## 常见问题

### API 连接失败

- 检查 API 密钥是否正确
- 确保 API 密钥有正确的权限
- 检查网络连接是否正常
- 确认 OKX API 服务是否正常

### 策略不执行

- 检查策略代码是否正确
- 确保策略已启动
- 检查市场数据是否正常获取
- 查看日志文件了解详细错误信息

### 数据导出失败

- 确保安装了 openpyxl 库（用于 Excel 导出）
- 检查文件保存路径是否有写入权限
- 确保磁盘空间充足

## 故障排除

### 查看日志

应用程序会生成日志文件，记录运行状态和错误信息。可以通过查看日志文件了解详细的运行情况。

### 性能问题

- 关闭不需要的标签页和功能
- 减少同时运行的策略数量
- 调整仪表盘卡片数量
- 检查系统资源使用情况

### 安全问题

- 不要分享 API 密钥和加密密钥文件
- 定期更新 API 密钥
- 使用强密码保护 API 密钥
- 只在安全的环境中运行应用程序
